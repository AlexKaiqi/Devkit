# 项目目标

## 愿景

打造一个 **AI 分身（希露菲）**，全权代理用户的开发、科研、智能家居和日常生活管理。用户通过手机语音或文字下达目标，希露菲拆解任务、调度工具、自主执行、汇报结果。

核心理念：
- **分身而非助手** — 有自主决策权，不是被动等指令的工具
- **AI 原生** — 项目本身的初始化、维护、扩展都由 AI 驱动
- **全链路本地** — 所有服务本地运行，数据不外泄

## 当前迭代目标

1. **外部工具链集成** — 邮件、日历、搜索引擎、云存储等基础连接能力
2. **AI 原生项目体验** — git clone 后 AI 即可引导完成全部初始化
3. **科研能力** — 论文搜索、文献管理、数据分析工具链
4. **App/GUI 操作** — macOS 桌面自动化 (peekaboo)、Web 自动化 (Playwright)

## 约束条件

- **平台**: macOS (Apple Silicon) + Cursor IDE
- **工具选型**: 开源优先、CLI 优先、闭源仅限不可替代
- **Python**: 3.12+，项目内 .venv 管理
- **容器**: Docker Compose 管理有状态服务
- **AI 无关性**: 不绑定特定 LLM，通过 OpenAI 兼容接口切换
- **中文优先**: 文档和沟通以中文为主，技术术语保留英文
