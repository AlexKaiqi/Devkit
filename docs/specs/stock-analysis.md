# 股票分析能力

## 背景

用户关注 A 股和港股市场，需要便捷地查询行情数据和财务信息。

## 需求

### A 股数据查询

通过 MCP 协议获取中国 A 股市场数据：

- 历史 K 线数据（分钟/小时/日/周/月/年级别）
- 核心财务指标（营收、净利润、资产等）
- 财务三表（资产负债表、利润表、现金流量表）
- 个股新闻与内幕交易信息
- 30+ 技术指标（MACD、RSI、布林带等）

### 使用场景

- "看看茅台最近走势" → 获取最近 K 线数据
- "贵州茅台的净利润是多少" → 查询财务指标
- "比较宁德时代和比亚迪的营收" → 多股对比

## 验收标准

- [x] akshare-one-mcp 通过 mcporter 端到端验证
- [x] 历史行情查询可用
- [x] 财务三表查询可用
- [ ] 港股数据接入

## 技术约束

- 通过 MCP 协议 + mcporter 调用
- 股票代码格式：纯数字（600519/000001/300750）
- 数据源：东方财富、新浪财经（沪市优先 sina）
