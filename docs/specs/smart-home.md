# 智能家居能力

## 背景

用户希望通过自然语言控制家中设备，实现智能化的家居管理。

## 需求

### 设备查询与控制

- "客厅灯开着吗" → 查询设备状态并回答
- "把空调调到 26 度" → 调用服务执行
- "关掉所有灯" → 批量操作
- 理解自然语言指令，映射到具体设备和操作

### 场景管理

- 维护场景列表（回家、离家、睡眠、起床、会客等）
- "我要睡了" → 执行睡眠场景
- 新建/修改场景：用户口述规则，Agent 转为自动化配置

### 环境监测

- 温湿度、空气质量、光照等传感器数据汇总
- 趋势分析和异常告警
- 智能建议（开窗通风等）

### 能耗管理

- 各设备/房间用电量统计
- 异常高耗电检测
- 月度能耗报告

### 安防监控

- 门锁状态查询
- 异常活动检测
- 离家模式自动布防

## 验收标准

- [ ] Home Assistant 连接配置完成
- [ ] 自然语言可查询设备状态
- [ ] 自然语言可控制设备

## 技术约束

- 全部通过 Home Assistant REST API 实现
- hass-cli 已安装，需要 HA 长期访问令牌

## 相关决策

- IoT 中枢选 Home Assistant（开源、生态最大、REST API 完善）
